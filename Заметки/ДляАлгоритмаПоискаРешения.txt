Нужно модифицировать алгоритм (выбора способа) решения Задачи так, чтобы он попутно создавал задачи для развития моей рабочей среды.
На этапе выбора способа решения Задачи выбрать один или несколько лучших способов и поставить в план Задачи по добыче ресурсов для систематического применения выбранных способов.  

Для решения некоторой задачи нужно:
1. Перечислить известные мне существующие теоретические способы решения данной Задачи.
2. Выбрать из этих способов наилучший по каким-то критериям, установленным как попало, на основе предыдущего опыта, настроения, предпочтений, итп.
- Для каждого способа решения задачи требуются ресурсы: оборудование, инструменты, расходные материалы.
- Выбор применяемого способа решения Задачи производится, в том числе, по наличию этих ресурсов для каждого метода, поскольку без таких ресурсов метод не может быть применен. 
3. Выполнять решение задачи выбранным способом, пока не будет достигнут результат, или пока выбранный способ не покажет свою неэффективность в данном случае.

Предлагается: в пункте 2 этого Плана оценивать существующие способы решения Задачи по их эффективности, экономичности, итп.
 Выбрать лучший способ, оценить доступность ресурсов для него и поставить в план задачу по добыче ресурсов для систематического применения такого лучшего метода в будущем.
 - Вряд ли там будет только один хороший способ,
 - Вряд ли возможно точно оценить свойства каждого способа, если его нельзя практически применить. 
    Следовательно, каждый способ надо вначале опробовать.
    Следовательно, надо приобрести необходимые ресурсы в достаточном количестве.
    И затем поддерживать это количество ресурсов.
   - тогда тут нужен прогноз - оценка частоты использования способа и расхода им ресурсов.
 - Значит, в итоге, все сводится к списку способов, значениям их свойств, списку ресурсов для каждого способа, сведений, опробован ли данный способ, итд.
 - Получится много теоретически возможных способов и один-два опробованных. 
   И по неопробованным способам можно будет создать много задач по подготовке к их опробованию.
    И по предварительной оценке свойств этих способов до их опробования.
   По сути, опробование нового способа решения задачи превращается в проект по улучшению моей рабочей среды.
   Это такой эксперимент, но по сложности, подготовке, исполнению и формальности сравнимый с проектом.
